### 🧭 On building responsible AI

The realistic risk in AI systems isn't a rogue superintelligence — it's quieter than that. A model trained on skewed data will confidently repeat the bias baked into it, and it will do so at scale. That's not a hypothetical for me; it's a design constraint I keep in view whenever a project touches ML — from *DisasterShield AI*'s prediction models to *AstraClear AI*'s image classification.

<sub><b>My working rule:</b> question the training data before you question the model, audit outputs for skew before you ship, and never let "the algorithm said so" replace human judgment.</sub>
